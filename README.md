## Overview
This is a multi-page personal portfolio website designed to display academic, personal, and project information. The website has been updated to support multi-platform viewing, including desktop, tablet, and mobile devices. It uses HTML and CSS only, with no frameworks.

## Pages Included
- **index.html** – Homepage with introduction, hero section, profile image, and highlights.
- **about.html** – Personal background, skills, interests, and experience.
- **projects.html** – List of completed projects with descriptions.
- **contact.html** – Contact information and availability.

## Features
- Multi-platform layout (desktop, tablet, and mobile friendly).
- Consistent dark neon theme across all pages.
- Clear navigation bar with page highlighting.
- Organized sections for readability.
- Profile and gallery images supported.
- Simple, clean HTML and CSS structure.
