# Extra-B Final Report  
**Project:** Personal Portfolio Website  
**Author:** Hoàng Minh Thống 
**Date:** 2025  

---

## 1. Introduction
This project is a multi-page personal portfolio website developed using basic HTML and CSS.  
Its purpose is to present personal information, academic background, projects, and contact details in a clean, structured format.

Originally, the design targeted desktop-only layout.  
However, the website has been updated to support **multi-platform viewing**, including desktop, tablet, and mobile devices.

---

## 2. Project Objectives
- Create a functional multi-page personal website.
- Organize information clearly using separate HTML files.
- Apply a consistent dark neon theme across all pages.
- Support multi-platform viewing through responsive layout adjustments.
- Ensure usability across different screen sizes.

---

## 3. Website Structure
The project contains four main pages:

### **1. index.html**
- Introduction and hero section  
- Profile image  
- Quick overview and highlighted skills  
- Showcase of recent work

### **2. about.html**
- Personal background and interests  
- Education and experience  
- Skills list  
- Additional descriptive sections

### **3. projects.html**
- Grid layout of project cards  
- Descriptions of each project  
- Example details and instructions for running certain projects

### **4. contact.html**
- Email  
- Phone number  
- Location  
- Availability information  

All pages share a consistent navigation bar and dark neon theme.

---

## 4. Design Choices
### **Color Theme**
A dark purple + neon color palette was selected for visual contrast and modern appearance:
- Neon magenta  
- Neon cyan  
- Neon yellow  
- Deep purple background  

### **Layout**
- Clean, modular section-based layout  
- Clear headers and spacing for readability  
- Images positioned to support descriptions  

### **Multi-Platform Support**
To ensure compatibility across devices:
- Width and spacing have been adjusted for different screen sizes  
- Elements scale proportionally when viewed on mobile  
- Navigation items wrap correctly  
- Text remains readable on small screens  

---

## 5. Technologies Used
- **HTML5** for structure  
- **CSS3** for styling  
No JavaScript or frameworks were used.

---

## 6. Conclusion
This project successfully meets all requirements for Extra-B:
- Four-page static website  
- Cohesive styling and layout  
- Multi-platform viewing support  
- Organized content with personal and project information  

The website is easy to navigate and visually consistent, making it suitable as a personal academic portfolio.

---

## 7. How to View
1. Extract or open the project folder.
2. Open **index.html** in a web browser.
3. Use the navigation bar to explore all pages.
